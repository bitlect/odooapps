Software License Manager

Manages your end user licence agreement

Software License Manager allows you to manage end-user licence agreements. You can configure EULA (End-user License Agreement), License, generate license key and validate it.

This bundle consists of the following modules:

  • Database Expiry - allows a customer database to expire at a given date.
  • License - Create a license, renew licence and generate invoice

Main Features:

  • Configure Database expiry date
  • Automatically expire database based on the expiry date
  • Generate License key
  • Generate Invoice
  • Renew License
  • Validate License key

The main idea:

  • Service provider activates a database with a licence key and sets expiry date
  • Database expires and requires user to renew
  • Service provider invoices the customer.
  • Customer pays for the invoice and the licence key is renewed
  • User inputs the licence key and renews the database
